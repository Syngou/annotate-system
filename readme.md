## 使用方法
* 下载源码
* 在命令行中进入到annotation文件夹中
* 在命令行中输入以下命令，会打开前端页面
```
npm install
npm run dev
```
* 如果要搞后端，还有再输入以下命令
```
npm run build
cd ..
python manage.py runserver
```
* 在浏览器地址栏输入127.0.0.1:8000
