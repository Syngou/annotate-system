# 医疗文本标注和处理系统
>   注意:  
>  本项目处于开发阶段，目前还用不了
>

```
目前界面惨不忍睹，后期会美化

```

## 环境要求:
<strong>python3.7及以上，django3 </strong> 

## 使用方法
> 1.   下载源码  (点击download zip即可)  
> 2.   进入到manage.py所在的文件夹  
> 3.   在命令行输入  python manage.py migrate  
> 4.  在命令行输入 python manage.py runserver  
> 5.   浏览器地址栏输入127.0.0.1:8000(注意，标点符号一定要是英文标点!)
